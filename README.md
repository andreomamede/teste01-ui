# Teste01-Client

O projeto foi gerado com [Angular CLI] version 8.0.1.

## Instalação

npm install
npm install -g @angular/cli

## Build do projeto

Executar `ng build` para build do projeto. Obs: será gerada a apasta dist/.

## Executar aplicação

Executar `ng serve`. Navegar na URL `http://localhost:4200/`.
